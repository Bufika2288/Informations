# Informations

  Displays usefull stuff like: Groups, Owner, Donate informations
  
  +Translate support

Commands:

    For steam group:
  
    sm_steamgroup
    sm_group
    sm_steam
  
    For discord group:
  
    sm_discord
    sm_dc
    
    For donate:
  
    sm_donate
    sm_support
    sm_don
  
    For owner profile:
  
    sm_steamprofile
    sm_profile
    sm_owner
    
    For TS Server:
    
    sm_ts3
    sm_teamspeak
    sm_ts
    
    For Vip information:
    
    sm_bvip
    sm_vipi
    sm_buy
    sm_vipinfo
    
Cvars:

    sm_steamgroup_enable
    sm_discord_enable 
    sm_owner_enable 
    sm_ts3_enable 
    sm_donate_enable
    sm_vipinfo_enable
    sm_informations_version
